# Front-end para gerenciar cardápios de produtos
-Este projeto lista cardápios categorias e produtos.

## Para rodar

Adicione um arquivo .env conforme arquivo .env.example

```bash
$ npm install
```
```bash
$ npm run build
```

```bash
$ npm run start
```
